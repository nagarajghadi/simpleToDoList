| Step | Command                      | Description                       |
| ---- | ---------------------------- | --------------------------------- |
| 1    | `git branch`                 | Check current branch              |
| 2    | `git switch main`            | Move to main branch               |
| 3    | `git pull origin main`       | Update main from GitHub           |
| 4    | `git merge feature-branch`   | Merge feature into main           |
| 5    | *(resolve conflicts if any)* | Edit → `git add .` → `git commit` |
| 6    | `git push origin main`       | Upload merged code to GitHub      |
| 7    | `git branch -d feature`      | Delete feature branch (optional)  |
